PwnIt Number Chain build hotfix

This hotfix replaces:
- src/games/number-chain/NumberChainGame.tsx

Why:
- The previous Number Chain file mixed `shownHints`/`totalHints` with the repo's actual progressive-run public challenge shape, which is `shownClues`/`totalClues` for the `clue-ladder` game key.
- That mismatch caused the Vercel TypeScript build failure.

What changed:
- Replaced the Number Chain component with a clean file that uses `shownClues` and `totalClues` consistently.
- Kept the live/verified flow on the existing `clue-ladder` backend game key so no DB remap is needed.
- Kept reveal/progress requests wired to `/api/attempt/progress`.
